# Tails

An ongoing personal project to develop a toy TLS 1.3 server, based on the
chapter "フルスクラッチして理解するSSL/TLS" of the book
"[エムスリーテックブック8](https://techbookfest.org/product/b94hFWewG7fVRLgqEmSjT1)".

![logo](./logo.png)

For reference, the chapter author's official code can be found at
[asmsuechan/tiny-tls](https://github.com/asmsuechan/tiny-tls).

Please note that this repository is a personal project and does not accept
issues or pull requests.
